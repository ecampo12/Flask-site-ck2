About this page

If you are seeing this page then that means everything is doing fine.

This site is being hosted on my raspberry pi at home. I set it up to act as a web server. The site you are viewing was created using Flask, a python based microframework. My server is running on ubuntu server, with nginx and gunicorn.


If there is not a weird domain name in your url, that means my personal domain name routed you to my server correctly.


Here are the resources I used to get this set up:

  • How to Make a Raspberry Pi Web Server
  • Corey Schafer's - Deploy to Linux Server
  • Corey Schafer's - Flask Tutorial Playlist
  • Pretty Printed's - Flask Tutorial
  • A Headless Raspberry Pi Home Server; a guide for those who choose not to connect a mouse and keyboard to their raspberry pi
  • Setting up a web server on a Raspberry Pi
  • Access your files from anywhere in the world with this web server made with a Raspberry Pi.

If you want you can get your own raspberry pi starter kit here.


**UPDATE**

As of July 2019, this flask site will no longer be hosted on my home raspberry pi webserver. I switched IPS, and they use Carrier Grade NAT. This means I will not be able to just forward some ports to make my server accessible. I am aware of some of the methods to get around a CG NAT, but I have not found one that I like.


For now, this flask app will be hosted on Heroku.
